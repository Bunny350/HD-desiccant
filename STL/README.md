This file only contains the default configuration of HD desiccant. It has the height of 60mm. To change the desiccant container / holder's height, you may use its CAD file and adjust the height. You can also [visit it in MakerWorld](https://makerworld.com/models/2565724-hd-desiccant-holder#profileId-2827543) to adjust the height to your filament's thickness online.

You can choose either one with the normal top lid or the foolproof top lid with the hexagonal top brim.
